# Utilities

Helper tooling that is not part of Luna's runtime. None of the core app (backend, frontend, electron, cli) imports from here.

| Folder | What it contains |
|---|---|
| [`docs/`](docs/) | Architecture, benchmarks, CLI and process reference docs |
| [`scripts/`](scripts/) | Build scripts, icon generation, packaging, Docker helpers, shortcuts |
| [`tests/`](tests/) | Test suite |
