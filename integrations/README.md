# Luna Integrations

Platform-specific add-ons and extensions. Each sub-folder is self-contained.

| Folder | Platform | Type |
|---|---|---|
| [`google-workspace/`](google-workspace/) | Google Workspace | Apps Script add-on |
| [`office/`](office/) | Microsoft 365 | Office Add-in (manifest + taskpane) |
| [`vscode/`](vscode/) | VS Code | Extension (TypeScript) |

## Contributing

Pick the folder for the platform you want to improve. Each has its own `README.md` with setup instructions. No changes to the core Luna backend or frontend are needed to work on these.
