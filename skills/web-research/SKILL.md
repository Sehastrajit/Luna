# Web Research

Use this skill when the user asks for current information, comparisons, source-backed answers, or website research.

Workflow:

1. Search the web when the answer may have changed recently.
2. Fetch or read relevant pages.
3. Compare sources before stating a claim.
4. Summarize directly and include the important source context.
5. Use a dynamic widget when the answer benefits from comparison, timeline, map, table, or visual explanation.
